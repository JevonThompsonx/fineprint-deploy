## Install 

```bash
powershell -command "Invoke-WebRequest -Uri 'https://raw.githubusercontent.com/JevonThompsonx/fineprint-deploy/main/install.bat' -OutFile '%temp%\install.bat'" && "%temp%\install.bat"
```

Uninstall

```bash
powershell -command "Invoke-WebRequest -Uri 'https://raw.githubusercontent.com/JevonThompsonx/fineprint-deploy/main/uninstall.bat' -OutFile '%temp%\uninstall.bat'" && "%temp%\uninstall.bat"
```
